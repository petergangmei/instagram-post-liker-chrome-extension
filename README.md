# Instagram Liker Chrome Extension

An automation tool that helps you stay engaged on Instagram by automatically liking posts and navigating to the next one.

## Features

- **Smart Detection**: Automatically identifies if a post hasn't been liked yet.
- **Auto-Navigation**: Clicks the "Next" button immediately after liking (or skipping if already liked).
- **Human-Like Behavior**: Includes random delays (3-6 seconds) between actions to mimic natural user interaction and help prevent being flagged as a bot.
- **Easy Control**: Simple Start/Stop interface from the extension popup.

## How to Install

1.  Open Chrome and navigate to `chrome://extensions/`.
2.  Enable **Developer mode** using the toggle in the top-right corner.
3.  Click the **Load unpacked** button.
4.  Select the folder where you saved this extension:
    `/Users/petergangmei/Developer/Crossplatform/instagramliker`

## How to Use

1.  Go to [Instagram](https://www.instagram.com).
2.  Open any post in "modal" view (click a post from the Explore page or a profile).
3.  Click the **Instagram Liker** extension icon in your browser toolbar.
4.  Click **Start Automation**.
5.  To stop the process at any time, open the popup again and click **Stop Automation**.

## Safety Warning

> [!WARNING]
> This tool is for educational purposes. Instagram has strict policies against excessive automation. Use this extension moderately to avoid temporary blocks or account restrictions.
